# 지뢰 찾기 구현하기
React를 구현한 지뢰찾기 게임입니다.
<br/><br/>

## 플레이방법
초기 화면은 다음과 같습니다.
지뢰는 😈, 타이머는 🕙로 표기됩니다. 또한 ⚙️를 클릭해 게임 설정을 할 수 있습니다.
<br/><br/>
